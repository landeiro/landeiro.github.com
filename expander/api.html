API

You can use directly the SUE API

http://landeiro.no.de/expand/?shorturl=http://bitly.com/jawLQE

the result will be 

{
  "shorturl":"http://bitly.com/jawLQE",
  "longurl":"https://github.com/landeiro/PJsonCouch",
  "milliseconds":47
 }

Handling errors

http://landeiro.no.de/expand/?shorturl=thisWillCauseAndError

the result will be 

{
  "shorturl":"http://thisWillCauseAndError",
  "longurl":"error",
  "message":"ENOTFOUND, Domain name not found",
  "milliseconds":613
}
